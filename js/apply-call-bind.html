apply,call,bind都是将一个方法上下文(也就是this值)设置为apply,call,bind的第一个参数,这个参数的对象没有这个方法

例子1-调用其他对象的方法

var girlone = {
name: "妞妞",
sex: "女",
age: "18",
hobby: "play,film",
makeup: function(){
	console.log(this.name+"会化妆")
    }
};
var boyone = {
	name: "冬冬",
	sex: "男"
};
console.log(girlone.makeup())
console.log(girlone.makeup.apply(boyone))
例1：妞妞有化妆方法，冬冬没有，可以用apply,call,bind将妞妞化妆方法的this值指定为冬冬，那么冬冬也就有化妆方法了。
